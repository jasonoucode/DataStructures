Problem:

A stack is called a last-in-first-out (LIFO) collection.
This means that the last thing we added (pushed) is the first thing that gets pulled (popped) off.
If the last card we put on our stack of cards was an ace,
then the first card we pulled from the top is that same ace.



Compiling:

$ make


Running/Testing:

$ make test


Cleaning .class files:

$ make clean
